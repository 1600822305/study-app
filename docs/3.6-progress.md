**4.1 极值定义与判定**（基础送分）：

经过多轮迭代，最终结构：

1. **概念 1：零点**（看原函数 <Math tex="f" />，蓝卡）
   - 课本标准措辞："使得函数值为零的自变量的值"
   - 几何意义 + 例子（<Math tex="f=x^2-1" /> 的零点 ±1）

2. **概念 2：驻点**（看导函数 <Math tex="f'" />，绿卡）
   - 课本标准措辞："函数一阶导数为零的点"
   - 等价关系单行表格：<Math tex="f" /> 的驻点 ⇔ <Math tex="f'" /> 的零点；<Math tex="f'" /> 的驻点 ⇔ <Math tex="f''" /> 的零点（高中不考）
   - 例子：<Math tex="f=x^3-3x" /> 的驻点 <Math tex="x=\pm 1" />

3. **概念 3：极值点 / 极值**（橙卡，含两个 Mafs 小图）
   - 极值点（x 坐标）：单调性变化的位置
   - 极值（y 值）：函数在极值点的函数值
   - 右侧两个小图（88×58）：⌒ 极大值（红 <Math tex="y=-x^2+2" />）+ ⌣ 极小值（蓝 <Math tex="y=x^2-2" />）
   - 经典例子：二次函数对称轴就是极值点；将对称轴代入 <Math tex="f(x)" /> 算出的函数值就是极值
   - 具体例：<Math tex="f=x^2-4x+3" />，对称轴 <Math tex="x=2" />，极小值 -1

4. **极值判定定理**（核心方法，灰卡）
   - 设 <Math tex="a" /> 是驻点，看 <Math tex="f'" /> 两侧正负
   - 表格三行：左正右负 → 极大值点；左负右正 → 极小值点；同号不变号 → 不是极值点
   - 一句话：驻点变号 = 极值点；驻点不变号 = 假极值点
   - 反例：<Math tex="f=x^3" /> 在 <Math tex="x=0" />

**关键认知澄清**（教学过程中反复迭代）：
- 三个概念**完全独立**：零点看 <Math tex="f" />，驻点看 <Math tex="f'" />，极值点看单调性变化
- **驻点 = 一阶导函数的零点**（同一个 <Math tex="x" />，两个名字）
- **极值点一定是驻点，反之不一定**（必须 <Math tex="f'" /> 变号）
- 拐点不考（涉及二阶导数）
