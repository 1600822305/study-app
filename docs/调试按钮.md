# 调试按钮说明

## 统一调试入口

项目已将三套调试系统（Mafs / Geo2dSvg / Geo3dSvg）统一为一个浮动按钮。

### 用法

在页面底部添加：
```tsx
import { UnifiedDebugToggle } from '@/components/shared';

// 在页面末尾（LessonLayout 外层 div 内）
<UnifiedDebugToggle />
```

### 功能

- **一键切换**：点击按钮同时开启/关闭所有调试模式（Mafs 函数曲线、2D 平面几何、3D 立体几何）
- **按钮位置**：右下角固定（`bottom-4 right-4`）
- **状态显示**：开启时显示 "📏 ON"（青色），关闭时显示 "📏 Debug"（白色半透明）
- **图上入口**：各图上的 🛠 小按钮保持不变（需先开启全局调试模式）

### 原有按钮

以下按钮仍保留导出，但建议新页面统一使用 `UnifiedDebugToggle`：
- `DebugToggle`（Mafs 专用）
- `Geo2dDebugToggle`（2D 专用）
- `Geo3dDebugToggle`（3D 专用）
- `GeoDebugToggle`（2D+3D 合并）

### 技术实现

- `UnifiedDebugToggle` 同时监听 `useDebugEnabled`（Mafs）、`useGeo2dDebug`、`useGeo3dDebug`
- 各调试系统内部仍使用各自的 `useSyncExternalStore` 单例模式
- 点击逻辑：任一调试开启则全关，否则全开
