## SVG 图形系统

- **Geo3dSvg**: 3D→2D 斜二测投影，用于立体几何（长方体、棱锥等）
- **Geo2dSvg**: 2D 平面几何，支持 angleArcs / tickMarks / rightAngles，用于初中几何复习
- 两套系统都有 Debug 调试面板（拖拽标签/顶点 + 复制代码）

## 规范

- `CONVENTIONS.md`: 10 个章节
- `@/` 路径别名
- ESLint + Prettier + TypeScript strict + erasableSyntaxOnly
- 数据驱动渲染：题目数据在 `features/xxx/data/`
- 存储层规则：组件 → Hook → `storage.xxx` → Repository → Dexie，禁止裸 localStorage
