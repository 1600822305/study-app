## 二、高考考点全景

### 2.1 定义层
- 定义：`a^x = N ⟺ x = log_a N`（条件 `a > 0, a ≠ 1, N > 0`）
- 三个名字：底数 `a`、真数 `N`、对数值 `x`
- 定义域：底数 `> 0` 且 `≠ 1`、真数必须 `> 0`
- 特殊符号：`lg N = log_{10} N`（常用对数）、`ln N = log_e N`（自然对数）

### 2.2 四个核心恒等式
1. `log_a 1 = 0`（真数为 1）
2. `log_a a = 1`（底数 = 真数）
3. `log_a a^n = n`（化幂为系数）
4. `a^{log_a N} = N`（对数还原指数）

### 2.3 三大运算法则
- 乘 → 加：`log_a(MN) = log_a M + log_a N`
- 除 → 减：`log_a(M/N) = log_a M - log_a N`
- 幂 → 乘：`log_a M^n = n · log_a M`

派生：
- `log_a √[n]{M} = (1/n) log_a M`
- `log_{a^n} M^m = (m/n) log_a M`

**关键能力**：法则的**正用**（拆开）与**逆用**（合并凑整）

### 2.4 换底公式
- 主公式：`log_a b = log_c b / log_c a`（c > 0, c ≠ 1）
- 高频变形：
  - `log_a b · log_b a = 1`（互为倒数）
  - `log_a b = 1 / log_b a`
  - `log_{a^n} b^m = (m/n) log_a b`
  - 链式：`log_a b · log_b c · log_c d = log_a d`（中间项相消）

### 2.5 求导（导数·提前接触）
- `(ln x)' = 1/x`
- `(log_a x)' = 1 / (x ln a)`
- 复合：`(ln f(x))' = f'(x) / f(x)`、`(log_a f(x))' = f'(x) / (f(x) · ln a)`

### 2.6 三条核心解题策略（贯穿全模块）

**策略 1：化简**
> 先用幂运算把底数 / 真数化成**分数指数幂**形式（让底数最简），再用对数法则化简合并。

**策略 2：逆用法则**
> 先化为同底对数的和差倍，再**逆用法则**，转回真数的积商幂。

**策略 3：指对互化**
> `a^x = N ⟺ x = log_a N`，是解决**一切对数问题**的核心方法。

> 这 3 条作为模块开篇导言卡，每组学完都要回头对照。

## 四、易错点与高考陷阱

- **真数必须 > 0**：`log_a M^2 ≠ 2 log_a M`（M 可能为负），严格写法是 `2 log_a |M|`
- **拆乘积陷阱**：`log_a(MN) = log_a M + log_a N` 要求 `M > 0 且 N > 0`，但原式只要 `MN > 0` 即可（M、N 可同负）— 拆分会缩小定义域
- **幂次拆分混淆**：`log_a(M^n)` ≠ `(log_a M)^n`，前者法则、后者乘方
- **底真都含 x**：`log_x(x^2 - 1)` 要同时满足 `x > 0`、`x ≠ 1`、`x^2 - 1 > 0`
- **lg / ln 混淆**：`lg = log_{10}`、`ln = log_e`，e ≈ 2.718
- **换底符号**：`log_{0.5} 3 = lg 3 / lg 0.5`，分母为负，结果取反
- **`a^{log_b N}` 不是 N**：恒等式要求底数与对数底数相同，否则要先换底
