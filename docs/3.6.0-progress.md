## 四、核心知识点

### 4.1 隐零点法（f' 全局单调时）

```
① 确定定义域，求导，发现 f'(x)=0 是超越方程
② 求 f''(x)，判断 f' 的单调性
   - f'' > 0 恒成立 → f' 递增
   - f'' < 0 恒成立 → f' 递减
③ 代入两个值看是否变号，确认零点存在，设为 x₀
④ 利用 f' 单调性 + f'(x₀)=0 判断两侧正负
⑤ 写单调区间
```

**代入值选择**：
- 指数题：x=0, 1（因为 e⁰=1, e¹=e 好算）
- 对数题：x=1, e（因为 ln1=0, lne=1 好算）

### 4.2 二阶导辅助法（f'' 有零点时）

```
f'' 有零点 → f' 有极值 → 通过极值判断 f' 的符号

- 最小值 > 0 → f' > 0 恒成立 → f 递增
- 最小值 = 0 → f' ≥ 0（临界）→ f 仍递增
- 最大值 < 0 → f' < 0 恒成立 → f 递减
- 最大值 = 0 → f' ≤ 0（临界）→ f 仍递减
```

### 4.3 原理：二阶导数决定一阶导数的单调性

```
f'(x) 的符号 → f(x) 的单调性（已知）
f''(x) 的符号 → f'(x) 的单调性（同理）
```
